AtmosAlert - Weather App

Languages Used: HTML, CSS, JavaScript

This weather application provides users with real-time weather information, a 5-day forecast, and air pollution data for any city worldwide. Users can search for a city or click on a location on the map to retrieve weather details, including temperature, humidity, wind speed, and weather conditions. Additionally, the app displays air quality index (AQI) data, highlighting the air quality level with color-coded alerts to indicate the severity.

Tic Tac Toe

Languages Used: HTML, CSS, JAVASCRIPT

Tic Tac Toe game is a web-based implementation of the classic two-player game. Designed with simplicity, it offers two modes of play: Play with Friend and Play with Computer.

Play with Friend: Two players can enter their names and take turns to compete against each other on the same device. The game keeps track of each player's wins and draws.

Play with Computer: A single player can enter their name and compete against a computer opponent. The game uses an AI algorithm to provide a challenging experience, with the computer making strategic moves.

The game features the current player's turn, a dynamic scoreboard that updates in real-time, and a winning screen. It also allows the user to either restart the game or exit the game.
